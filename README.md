# Lebenslauf (One-Page)

Statische One-Page-Website fuer den Lebenslauf von Viet Tri Nguyen.

## Inhalt
- `index.html`: die komplette One-Page-Seite
- `Lebenslauf_Zertifikate.pdf`: Referenz-PDF

## Lokal ansehen
- Windows: `start index.html`

## GitHub Pages (optional)
1. Repository Settings -> Pages
2. Source: `Deploy from a branch`
3. Branch: `main` / Folder: `/ (root)`
4. Save und die URL abwarten

Tipp: Wenn du spaeter eine eigene Domain willst, kann ich eine `CNAME` Datei anlegen.
